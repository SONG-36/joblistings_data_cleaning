# Field Missing Rate Summary

This document summarizes missing value patterns across key fields.

## Salary Fields

- salary_raw missing rate: ~54%
- salary_period explicit rate: ~44%
- salary_period missing rate: ~56%

## Interpretation

Salary missingness is systemic and correlates with job category and company,
independent of data processing logic.

## Decision

Missing values are treated as real-world phenomena, not data errors.
